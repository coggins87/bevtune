# Bevtune

This project utilizes the CocktailDB and Spotify's API.

Search for a cocktail, return ingredients, and create playlists based on the cocktail name!

In order to search and generate playlists, you need to login with your Spotify account.

This project was bootstrapped with [Create React App](https://github.com/facebook/create-react-app).
